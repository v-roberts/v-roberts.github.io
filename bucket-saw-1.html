Bucket Saw Full Assembly

This is the full assembly of our bucket saw. It includes the top plate, bracket, and footswitch base on a transparent example bucket.

We decided to attach a momentary footswitches so that, after it is plugged in, there is a faster, more responsive on / off control in place. The footswitch base is designed to be screwed to the bucket and have a footswitch screwed to it, so that when the saw is in use it will both require constant input from the user (pushing the pedal down) and will be harder to knock over.
